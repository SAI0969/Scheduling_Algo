#  CPU Scheduling Algorithms & Interactive Simulator

A comprehensive, interactive web-based educational resource for understanding and mastering CPU scheduling algorithms used in Operating Systems.

##  Live Website

**Visit the interactive simulator:** [https://sai0969.github.io/Scheduling_Algo/](https://sai0969.github.io/Scheduling_Algo/)

Experience the full interactive CPU scheduling simulator with real-time Gantt chart generation and visual process execution!

##  Overview

This project provides an in-depth exploration of **CPU scheduling algorithms** with detailed explanations, real-world examples, interactive simulations, and hands-on practice problems. Whether you're a student preparing for exams or a developer looking to understand OS concepts, this resource covers everything you need.

##  Features

###  Educational Content
- **Introduction to CPU Scheduling** - Understand the fundamentals and why scheduling matters
- **Key Terminology** - Arrival Time, Burst Time, Turnaround Time, Waiting Time, Response Time, and more
- **Scheduling Criteria** - Learn about throughput, CPU utilization, and performance metrics

###  Scheduling Algorithms Covered

1. **First Come First Serve (FCFS)**
   - Non-preemptive approach
   - Simple FIFO queue-based implementation
   - Multiple worked examples with Gantt charts
   - Convoy effect explanation

2. **Shortest Job First (SJF)**
   - Both preemptive and non-preemptive variants
   - Shortest Remaining Time First (SRTF)
   - Optimal average waiting time
   - Step-by-step execution breakdowns

3. **Priority Scheduling**
   - Preemptive and non-preemptive modes
   - Starvation problem and aging solution
   - Real-world priority handling scenarios

###  Interactive Features
- **Live Gantt Chart Generator** - Visualize process execution timelines
- **Interactive Simulator** - Input your own process data and see scheduling in action
- **Visual Process Queues** - See how processes move through the system
- **Question Generator** - Test your knowledge with auto-generated problems

###  Learning Resources
- **Gantt Charts** - Visual timelines showing CPU allocation
- **Comparison Tables** - Side-by-side algorithm performance metrics
- **Step-by-Step Breakdowns** - Detailed execution walkthroughs
- **Practice Problems** - Solve real problems and verify your understanding
